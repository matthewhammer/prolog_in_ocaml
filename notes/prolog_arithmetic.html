ISO Prolog: Arithmetic Operators and Functions

This table lists the standard arithmetic operators and functions in ISO Prolog, along with a tiny example of usage for each.

Category Functor / Operator Description Example

Binary Operator - Subtraction X is 5 - 2. % X = 3

Binary Operator / Floating-point division X is 5 / 2. % X = 2.5
Binary Operator // Integer division X is 5 // 2. % X = 2
Binary Operator rem Integer remainder X is 5 rem 2. % X = 1
Binary Operator mod Modulus X is 5 mod 2. % X = 1
Binary Operator ** Exponentiation X is 2 ** 3. % X = 8
Binary Operator ^ Exponentiation (synonym for **) X is 2 ^ 3. % X = 8
Binary Operator /\ Bitwise AND (integers) X is 6 /\ 3. % X = 2
Binary Operator \/ Bitwise OR (integers) X is 6 \/ 3. % X = 7
Binary Operator xor Bitwise XOR (integers) X is 6 xor 3. % X = 5
Binary Operator < Left shift (integers) X is 3 << 2. % X = 12
Binary Operator >> Right shift (integers) X is 12 >> 2. % X = 3
Unary Operator + Unary plus X is +5. % X = 5
Unary Operator - Unary minus X is -5. % X = -5
Unary Operator \ Bitwise complement (integer) X is \2. % X = -3
Unary Function abs/1 Absolute value X is abs(-7). % X = 7
Unary Function sign/1 Sign (-1,0,1) X is sign(-10). % X = -1
Unary Function sqrt/1 Square root (returns float) X is sqrt(9). % X = 3.0
